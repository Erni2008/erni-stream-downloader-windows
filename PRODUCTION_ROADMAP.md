# Production Roadmap

Идеи, которые стоит добавить перед публичным релизом:

- Code signing для Windows, чтобы SmartScreen меньше ругался на установщик.
- Автоматические GitHub Releases с `.exe`, installer и changelog.
- Отдельный экран настроек: папка по умолчанию, язык, тема, лимит одновременных загрузок.
- Проверка обновлений самого приложения.
- История загрузок с кнопками `Открыть файл`, `Открыть папку`, `Повторить`.
- Поддержка playlist-mode только для собственных плейлистов, с явным подтверждением пользователя.
- Детальный preflight: размер после конвертации, свободное место на локальном и внешнем диске.
- Интеграция `ffprobe` для проверки готового файла: есть ли видео, есть ли звук, codec, FPS, resolution.
- Crash reports/log export: одна кнопка, которая собирает `app.log` и параметры системы в zip.
- Полноценная локализация RU/EN.
- Автотесты для формирования команд, парсинга прогресса и обработки ошибок.
